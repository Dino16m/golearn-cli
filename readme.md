# Golearn CLI
### Command line tool for the [Golearn](https://github.com/dino16m/golearn) framework
---
## Installation

```bash
go install github.com/dino16m/golearn-cli
```

## Usage

### Setup a new web project

```bash
golearn-cli install <project>
```
